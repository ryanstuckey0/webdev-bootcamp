font stacks

  • we can also specify multiple fonts, with the fonts being prioritized from left to write
  • so specifying "verdana, sans-serif" will prefer verdana but choose a sans-serif font if verdana is unavailable
  • so we can create a "stack" of fallbacks if the chosen one is unavailable

font embedding

    we can go to fonts.google.com to look at various fonts and select them to embed in our webpage

    we can then add these fonts into the head section of the html document to use these fonts in our webpage
