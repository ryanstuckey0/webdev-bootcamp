How can we center all the elements on a page

We can use property text-align, which must be sent in the parent container.

This works as long as we have inline-block elements or full-width block elements.

But, if we have a block that does not take the full width, then it will not be centered.

We can fix this by adding a margin CSS property. "margin: 0 auto 0 auto" will center the element.

Auto centers the margin in either the horizontal or vertical direction.

This is not centered because it has a width set.
But this is centered because it has a width set and margin set.

Margin can be set as "top right bottom left"

Or "top right&left bottom

Or "top&bottom right&left
