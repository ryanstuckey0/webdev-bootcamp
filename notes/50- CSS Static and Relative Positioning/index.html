Rules for how HTML is displayed on webpage without CSS

  1. Content is everything-- Inline elements only take up as much space as the content. Block element height is determined by content height.
  2. Order comes from the code-- Content gets rendered in the order that the code is written.
  3. Children sit on top of parents-- In 3D space, children will be "closed to the viewer" than their parents. This determines which elements will be displayed in front of the others.

CSS Position Property

Static: All HTML elements are static by default, which means go along with default HTML flow.

Relative: Position an HTML element relative to how it would've been positioned in static.

We can use the left, right, top, and bottom properties to move the element around.

Moving an element with the relative property does not affect any element on screen.

The other elements act like it is still in it's original position.
